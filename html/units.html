Astropy Units Cheatsheet

Your shortcut for dealing with pesky unit conversions

I'm constantly telling people how much I love Astropy units so I figured I would make a quick cheatsheet to make starting to use them easier!

Importing
You'll need to this to use Astropy units
                                import astropy.units as u
                                
import astropy.constants as const
Applying units
Multiply by a unit to add it to a variable
some_mass = 72 * u.kg
                                
some_velocity = 265 * u.km / u.s
Unit conversion
Convert between any units using .to()
age = 24 * u.year + 300 * u.day
                                
age_in_fortnights = age.to(u.fortnight)
Removing units
Get the .value of a unit to convert it back to a number
some_time = 13.7 * u.Gyr
                                
just_the_value = some_time.value
Plotting units
If you want to plot something (say with matplotlib for instance), then you'll need to add this to the top of your file. 
from astropy.visualization import quantity_support
                                
quantity_support()
Simplifying units
The .decompose() method can simplify your units
something = 10 * u.mg / (100 * u.J)
                               
something_else = 5 * (u.g / u.cm**3) / (10 * u.km / u.yr)
                               
lots_of_units = something * something_else * u.erg
                               
lots_of_units.decompose()

Common Errors and Pitfalls
If you're anything like me you'll probably run into these
(10 * u.Msun).to(u.s)
                                
UnitConversionError: "'solMass' (mass) and 's' (time) are not convertible"

If you see this one then you've probably got a typo somewhere because it's not possible to convert between these units

1e5 * u.J / 5 * u.Myr    # this gives 20000 J Myr :(
                                
1e5 * u.J / (5 * u.Myr)  # this gives 20000 J/Myr :D

Parentheses are important! Be very careful or you'll probably get UnitConversionErrors because of this one

has_units = 10 * u.s
                                
no_units = 2
                                
has_units - no_units
                                
UnitConversionError: "Can only apply 'subtract' function to dimensionless quantities when..."

Don't mix your variables with units and those without, astropy will get mad at you!

A small working example

Here's a short little example of using astropy units (using everything above)

import astropy.units as u
                            
import astropy.constants as const
                            
import numpy as np
                            
import matplotlib.pyplot as plt
                            
from astropy.visualization import quantity_support
                            
quantity_support()
                            

                            
plt.rc('font', family='serif')
                            
plt.rcParams['text.usetex'] = False
                            
fs = 24
                            

                            
# update various fontsizes to match
                            
params = {'figure.figsize': (12, 8),
                            
          'legend.fontsize': 0.5 * fs,
                            
          'axes.labelsize': fs,
                            
          'xtick.labelsize': 0.6 * fs,
                            
          'ytick.labelsize': 0.6 * fs,
                            
          'axes.linewidth': 1.1,
                            
          'xtick.major.size': 7,
                            
          'xtick.minor.size': 4,
                            
          'ytick.major.size': 7,
                            
          'ytick.minor.size': 4}
                            
plt.rcParams.update(params)
                            

                            
def keplers_3rd(f_orb, m_1, m_2):
                            
    return (const.G * (m_1 + m_2) / (2 * np.pi * f_orb)**2)**(1/3)
                            

                            
n_binaries = 10000
                            

                            
f_orb = np.logspace(-5, -1, n_binaries) * u.Hz
                            
m_1 = np.ones(n_binaries) * u.Msun
                            
m_2 = np.ones(n_binaries) * u.Msun
                            

                            
a = keplers_3rd(f_orb, m_1, m_2)
                            
print(a.decompose())
                            
print(a.to(u.angstrom))
                            
print(a.to(u.km).value)
                            
 
                            
for i in range(1, 50 + 1, 10):
                            
    a = keplers_3rd(f_orb, m_1 * i, m_2).to(u.AU)
                            
    plt.scatter(f_orb, a, label=m_1[0] * i)
                            
plt.xscale("log")
                            
plt.yscale("log")
                            
plt.xlabel(f"Orbital Frequency [{f_orb.unit}]")
                            
plt.ylabel(f"Separation [{a.unit}]")
                            
plt.legend(title="Primary Mass")
                            
plt.show()
